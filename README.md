# QA Automation Python Sample

This repository demonstrates a simple QA automation testing setup using Python and pytest.

## Purpose

The goal of this project is to showcase a basic automated testing structure that can be used as part of a QA workflow.

## Technologies

- Python
- Pytest
- Requests

## Test Example

The test included verifies that a public API endpoint responds successfully.

## How to Run

Install dependencies:

pip install -r requirements.txt

Run tests:

pytest
